Tabela de Linguagens de Programação

Fiz uma pesquisa baseada em popularidade, facilidade e performance de cada uma das linguagens solicitadas e o resultado foi esse:

Linguagem Popularidade Facilidade Performance Opinião
Javascript Alta Média Boa Javascript(JS) é uma linguagem com uma popularidade muito alta atualmente no mercado e de fato é uma linguagem que da pra fazer de tudo principalmente em questão de front-end(client-side), pois hoje em dia o JS já faz parte de 97,8% de todos os sites no mundo de acordo com o site hostinger.
Possuo conhecimento básico em JS e na minha experiencia eu achei o JS uma linguagem fácil de se aprender.
Java Média Média Boa Java é uma linguagem que eu bato o olho e não me agrado muito, mas eu sei a necessidade dela no mercado e sua importância em sistemas, back-end, aprendizado em POO, seus framewoks bem elaborados e etc.
O conhecimento que eu tenho em Java foi aprendendo o básico de programação orientada a objetos.
PHP Baixa Fácil Média O php assim como o JS faz parte de maior parte dos nossos site, porém o php é uma linguagem de scripting focado no server-side, nos ultimos anos ela estava bem abaixo das outras mas acredito que com essas ultimas atualizações da 8.3 que vieram com várias funcionalidades novas e etc ela possa voltar a ser popular como antes. Nunca aprendi ou mexi com ph, mas tenho grande interesse na linguagem por ela ser uma linguagem bem forte em back-end(server-side).
Python Alta Fácil Boa Python para mim, é uma linguagem muito fácil e confortável de se utilizar e muito boa mesmo em vários quesitos, principalmente quando envolve Data Science e Automação, fiz um intenssivão em python e utilizei um pouco dessas ferramentas e automação em python é uma coisa muito magica e legal de se utilizar. Nessa lista python é outra linguagem que eu tenho muita vontande de aprofundar-me.
C# Média Média Boa C# é uma linguagem muito boa para aplicativos windows e jogos na minha visão e ela não se limita apenas a isso ela pode fazer back-end, machine learning e varias outras coisas, junto ao .NET ela fica ainda mais poderosa. Acho ela de fácil entendimento e aprendizado principalmente para quem já estudou C ou Java.
Rust Baixa Difícil Excelente Rust para mim atualmente é a linguagem que eu tenho mais interesse em me especializar, mas no momento ela não está tão forte no mercado, mas está em processo de crescimento no mercado, se formos analisar as últimas pesquisas de popularidade do stack overflow podemos ver o grande crescimento da comunidade que está utilizando o rust para várias coisas. Ela tem performance e segurança como seus destaques e isso é muito bom para alguém como eu que quer seguir uma carreira na área de cybersecurity além de que falam que ela é como se fosse um C++ melhorado ou que ela pegou o melhor de todas as linguagens e juntou todos em uma só.
Acho que se eu for dar todos os detalhes técnicos e features dessa linguagem vai ficar um texto gigantesco então deixo para você professor dar uma pesquisada nessa linguagem que muitas pessoas da nossa comunidade de programadores amam tanto.
